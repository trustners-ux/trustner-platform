Input Columns (blue/yellow):
- Column C: Total AUM/Book (manual entry)
- Column D: Trail Rate % (default 0.5%, editable)
- Column F: Source Type (dropdown selection)
- Column G: Incremental New Business (manual entry)

Formula Columns:
- Annual Trail = AUM * Trail Rate / 100
- Effective Base = IF formula based on Source Type
  * If Self-Sourced: Use AUM
  * If Assigned + New Biz: Use Incremental New Business
  * If Office/Walk-in: Use AUM
  * Otherwise: 0
- RM Share % = IF formula based on Source Type (25%, 15%, 0%, 10%)
- RM Annual Trail = Effective Base * Trail Rate / 100 * RM Share %
- RM Monthly Trail = RM Annual Trail / 12
- Mgmt Share % = IF formula based on Source Type (15%, 10%, 5%, 10%)
- Company Share % = 1 - RM Share % - Mgmt Share %

================================================================================
SUPPORT & TROUBLESHOOTING
================================================================================

Common Issues:

Q: Dropdown not working in Trail Income Model
A: This has been FIXED in v2. If still issues:
   - Refresh Excel (Ctrl+Shift+F9)
   - Re-open file (sometimes required after creation)
   - Verify cell is not protected

Q: Formula showing error
A: Check that source cells have data. Most formulas have IF statements
   for zero-division errors.

Q: Performance Status not showing
A: Enter some data in Weighted Business column first (column W).
   Status depends on Achievement %.

Q: Trail formulas giving wrong result
A: Verify Source Type dropdown is selected (not blank).
   Different source types use different calculations.

Q: FP Route 125% credit not calculating
A: Go to "Maker-Checker FP Log" and ensure:
   1. All 4 FP fields marked "Yes" (D:G)
   2. Manager Verification = "Verified" (I)
   3. Then 125% Credit Eligible will show "YES" (K)
