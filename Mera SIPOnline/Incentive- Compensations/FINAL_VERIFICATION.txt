TRAIL INCOME MODEL DROPDOWN FIX

Problem Description:
  The previous version had non-functional dropdowns in column F (Source Type)
  of the Trail Income Model sheet. This prevented users from selecting the
  business source, which is critical for calculating trail sharing percentages.

Root Cause:
  Dropdowns were either not created or not properly applied to individual cells
  using the openpyxl DataValidation class.

Solution Implemented:
  Created proper DataValidation objects using openpyxl with:
  - Formula: "Self-Sourced,Assigned + New Biz,Assigned + No New Biz,Office/Walk-in"
  - Error Title: "Invalid"
  - Error Message: "Select source type"
  - Applied to EVERY row (rows 3-40+) individually

Code Implementation:
  dv_source = DataValidation(
      type="list",
      formula1='"Self-Sourced,Assigned + New Biz,Assigned + No New Biz,Office/Walk-in"',
      allow_blank=True
  )
  dv_source.error = "Select source type"
  dv_source.errorTitle = "Invalid"
  ws_trail.add_data_validation(dv_source)
  
  for r in range(3, 42):  # All 37 employee rows
      dv_source.add(ws_trail.cell(row=r, column=6))

Impact:
  - Fixes: Trail income model can now be used
  - Benefit: Correct trail sharing calculations per source type
  - Result: All 37 sales employees can independently select source type
  - Verification: 37 separate dropdown instances confirmed
