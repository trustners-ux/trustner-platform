================================================================================
TROUBLESHOOTING
================================================================================

Q: Dropdown not working in Trail Income Model?
A: 1. Make sure you're in Column F (Source Type)
   2. Click the cell and look for dropdown arrow
   3. Press Alt+Down to open dropdown manually
   4. Try Ctrl+Shift+F9 to refresh
   5. If still not working, re-open the file

Q: Formula showing #DIV/0! error?
A: This is expected if source cells are empty
   Enter data in the input cells first
   Errors should disappear once you have data

Q: Performance Status not showing?
A: You need to enter business data first
   The status depends on Achievement %
   Achievement % depends on Weighted Business

Q: Trail formulas giving wrong result?
A: Check that Source Type is selected (not blank)
   Different source types use different RM share %
   If Source Type is blank, formulas return 0

Q: Can't find a particular employee?
A: Use Ctrl+F to search
   Go to Employee Master sheet first
   All 61 employees are listed there
